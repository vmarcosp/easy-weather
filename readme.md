# Easy Weather

Aplicação para consultar a previsão do tempo diária e semanal.

## Pré requisitos

- NodeJS v6+
- npm v5+

## Install

```sh
npm install
```

## Up & Running
Dev
```sh
npm run dev
```
## Demo server
```sh
npm run start
```
## Build
```sh
npm run build
```
